# Router  
LAN ile WAN arasındaki bağlantıyı sağlayan cihaz. Default Gateaway olur ve IP adresiyle çalışır. İnternet çıkışı var.

# Switch
Aynı ağdaki cihazları birbiriyle konuşturmasını sağlar. Örneğin bilgisayar ile yazıcı arasındaki iletişim. Şu MAC adresi şu porttaydı, veriyi oraya gönderiyorum der. İnternet çıkışı yok.

# Acces Point (AP)
Kablosuz cihazları kablolu ağa bağlar yani kablosuz Switch görevi görür. Telefon Wi-Fi'ye bağlandığında bağlantıyı Switch'e aktarır. İnternet çıkışı yok.

# Önemli Not:
Bu 3 kavramın hepsi çoğu modern modemin içinde aynı anda bulunur.
